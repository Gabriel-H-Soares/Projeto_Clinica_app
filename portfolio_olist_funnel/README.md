# Olist Funnel Analysis (Portfolio Project)

This repository section contains a portfolio project focused on analyzing and visualizing the Olist sales funnel.

## Project Overview

- `Project_olis_funnel.ipynb`: Jupyter Notebook with the exploratory analysis workflow, metrics investigation, and business-oriented insights.
- `streamlit_olist_funnel.py`: Streamlit app that turns the analysis into an interactive dashboard for practical exploration.

## Purpose

This is a portfolio project created to demonstrate end-to-end data analysis skills, from notebook-based exploration to an interactive application for stakeholders.
